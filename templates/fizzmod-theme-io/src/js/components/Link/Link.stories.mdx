# Link

## Usage

```
import Link from 'components/Link';

<Link href="/">Link text</Link>

<Link href="/" target="_blank" rel="noopener noreferrer">Link with target blank</Link>

<Link href="/">
    <img src="via.placeholder.com/200x200" alt="test image" title="text image" /> 
</Link>
```
